== Description ==
Instant Suggest Plugin is an editor plugin integrated with TinyMCE or CKEditor. This plugin can be used when you add new or edit an existing page or post.

= Features =
* Find SEO targeted keywords using <strong>Keyword Research Tool</strong>
* Find related articles to get ideas for your article using <strong>Article Search Tool</strong>
* Replace a term in your article with the keywords suggested by <strong>Keyword Suggestions Context Menu</strong>
* InstantSuggest Plugin works with both popular rich text editors: <strong>CKEditor</strong> and <strong>TinyMCE</strong>

= Why should you use Instant Suggest Plugin? =
* To find SEO targeted keywords
* To get content or idea for your web site
* To correct the phrases that you might make typing mistakes
* To autocomplete the phrase that you're typing in

== Installation ==

1. Upload the plugin to your '/wp-content/plugins/' directory, or download and install automatically through your admin panel.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Add new or edit existing post with InstantSuggest plugin integrated in the rich text box editor
